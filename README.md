## Setup Instructions

1. Create a new GitHub repository named `assets`
2. Make it **Public**
3. Upload all files from this folder
4. Go to **Settings** → **Pages**
5. Set Source: **Deploy from a branch** → **main** → **/ (root)**
6. Save and wait ~2 minutes
7. Your site will be live at `https://YOUR-USERNAME.github.io/assets/`
